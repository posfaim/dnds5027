What is the use of comparisons? So far all we wrote programs where each line was executed one-by-one without exception.

Therefore, instead of executing each line in sequence, the if-else pair introduce a branch in our program.







Some jargon: The order in which the commands are executed in a program is called control flow.
The conditional statement (i.e., the if-else pair) is a control flow statement, meaning that it affects which commands are executed.
You will soon learn more control flow statements.
