Strict naming rules (syntax):

    

  •  Can only contain alphanumeric characters (a-z, A-Z, 0-9) and underscores (_)
    • 

  •  Cannot start with a number
    • 

  •  Is case sensitive
    • 





Good practice (style guide):

    

  •  Give concise but descriptive variable names

  •  Follow recommended formatting, for example, snake case: favorite_food
