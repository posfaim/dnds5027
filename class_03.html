User defined functions



The purpose of functions is to re-use code: if there is a task that you have to repeat many times, you don't have to type the code over and over again.



Functions are also useful to organize your code into logical units. For example, having a separate function for printing means that the code for counting the number of characters in a string doesn't mix with the code needed for printing. This makes the code easier to read, write and debug.



The functions input, len and print are built-in functions, meaning that they come with Python, you can just straight use them. It is also possible to define your own functions.
